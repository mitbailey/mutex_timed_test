# Mutex Timed Test

Runs a number of tests to time mutexes in GCC compiled C, G++ compiled C, and G++ compiled C++.

## Usage

GCC Compiled C  
`make cmain`

G++ Compiled C  
`make ccmain`

G++ Compiled C++  
`make cppmain`
